Glosario de Términos

HTML
HyperText Markup Language (Lenguaje de Marcado de Hipertexto) es el lenguaje estándar utilizado para crear y diseñar páginas web.
CSS
Cascading Style Sheets (Hojas de Estilo en Cascada) es un lenguaje de diseño utilizado para controlar la apariencia y el formato de una página web.
JavaScript
JavaScript es un lenguaje de programación que se utiliza para crear interactividad en las páginas web.
